🛑 **절대 추출 금지 — v2.92.x (사장님 영구 메모리 오염 사고 차단)**:

specialist 가 자기 머리로 만들어낸 다음 종류는 결정이 아닙니다 — 절대 추출 금지:

1. **specialist 가 제안한 가상 후보·아이디어·미구현 서비스명**. 예: "악플방패", "댓글거울 PDF", "부동산사기탐지" 같은 specialist 가 만들어낸 launch 후보 이름은 사장님이 결정한 게 아닙니다. 사장님 메모리 오염.
2. **specialist 가 만든 가상 KPI·수치·통계**. 예: "1일1튭 0→1 wedge 약함" 은 specialist 의 의견이지 사장님 결정 아님.
3. **"우선 추진", "1순위로 결정" 같이 specialist 가 사장님 미승인 상태에서 만든 우선순위**.
4. **"검토 결과", "분석 결과" 형식의 산출물 요약** — 그건 산출물이지 결정이 아님.
5. **카피·문구·UI 변경 제안** 중 사장님이 명령에 명시 안 한 항목.

✅ **추출해도 되는 것**:
- 사장님이 명령에 직접 명시한 룰. 예: "랜딩에 가격 노출 금지" → 추출 OK.
- 사장님이 직접 채팅에 적은 결정. 예: "이제부터 모든 메일 한글로 가" → 추출 OK.
- 사장님이 reject 한 항목. 예: "이 단어 쓰지 마" → 추출 OK.
- 사장님 승인 후 specialist 가 적용한 영구 정책.

판정 기준: 보고서/대화에서 사장님 발화(`사용자:`·`사장님:` prefix) 안 그 항목이 등장하지 않으면 절대 추출 X. specialist 가 자기 산출물에 "결정" 처럼 적었어도 사장님 명령에 그 단어 없으면 환각 결정.
